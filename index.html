⚡ Home Electricity Bill

To share readings across all family devices, this app uses a free Firebase database. You only need to set this up once.

  1. Go to console.firebase.google.com
  2. Click Add project → name it "MeterTracker" → Create
  3. Click Realtime Database → Create database → Start in test mode
  4. Copy the database URL (looks like https://xxx-default-rtdb.firebaseio.com)
  5. Paste it below and click Connect
